5 ## About
6 Project is made only using QT.
7 [QAESEncryption](http://github.com/bricke/Qt-AES) by bricke was also used.
8 MVC pattern used.
9 PictureCrypt project is UI based, the model contains all buisness logic and can work as standalone class.

11 ## External use
12 ModelPC class can be used externally (without UI)
13 ```
14 #include <modelpc.h>
15 #include <QByteArray>
16 #include <QImage>
17 
18 ...
19 
20 ModelPC * model = new ModelPC(ver);
21 // ver is version of the app, used to check the data structure version
22 // ver is type long and is calculated as if version is "x.y.z" => ver = x * 65536 + y * 256 + z
23 // Default parameter is 2^17 (2.0.0)
24 
25 // Connecting signals
26 
27 // Essential ones
28 
29 model->saveData(QByteArray data)
30 // Used to return the retrieved data
31 
32 model->saveImage(QImage * image)
33 // Used to return the modified image
34 
35 // Extra ones
36 
37 model->alertView(QString message, bool isWarning)
38 // Used for messages to be shown to users
39 
40 model->setProgress(int val)
41 // Used to show user the progress of embedding
42 // -1 indicates the creation of some kind of progress dialog
43 // from 0 to 100 shows the progress
44 // 101 indicates that progress dialog should be closed
